做噩夢了。 
夢到那天，把第三隻眼閉起來的時候。

夢裡的聲音很吵雜，比地靈殿最深處的岩漿還要滾燙，比怨靈的嘶吼還要刺耳。
那是「人心」的聲音。
恐懼、嫉妒、厭惡、算計。
無數條由意念編織成的荊棘，透過那顆睜得大大的眼睛，毫無保留的刺進我的心裡。

『為什麼要讀我的心？』『好噁心。』『別過來，妖怪。』

夢裡的我看見了姐姐。
姐姐總是皺著眉頭，默默承受著那些無聲的謾罵。
雖然姐姐什麼都沒說，但我聽得到。

姐姐的心在哭。

因為我們能看透一切，所以我們被一切所拒絕。

這份「天賦」，其實是劇毒。

在夢裡，我再次做出了那個決定。
我伸出手，指尖觸碰到胸前那顆敏感、顫抖著的眼球。
它瘋狂地轉動著，接收著來自四面八方、充滿惡意的訊息。

「只要看不見就好了。」「只要聽不到就好了。」「只要... 什麼都不想，就不會受傷了。」

噗滋。

那不是肉體被撕裂的聲音，而是靈魂連結被強行扯斷的聲響。
劇烈的疼痛像電流一樣竄過全身，視野瞬間染成一片血紅，隨即陷入絕對的漆黑。
原本在腦海中轟鳴的萬千雜音，在剎那間——

死寂。

世界安靜得像是一口枯井。

那些愛、那些恨、那些因為「理解」而產生的痛苦，全部被擋在了眼瞼之外。

...

它沒有張開。太好了。

剛才的噩夢是什麼感覺來著？恐懼嗎？悲傷嗎？我歪了歪頭，試圖回憶殘留的情緒。
但是，那些情緒就像水面上的泡沫，剛浮現就啵的一聲消失了。
現在的我，讀不到別人的心，也讀不懂自己的心。

我什麼都不知道。

「戀？妳在這裡做什麼？」

身後傳來姐姐的聲音。
我回過頭著那張熟悉的臉龐，是姐姐。

但現在我不知道她在想什麼。
我也不需要知道她在想什麼。

我只感覺到，嘴角不自覺地揚起。
露出一個空洞卻快樂的笑容。

「沒事喔，姐姐。」

我轉過身，像一陣抓不住的風，蹦蹦跳跳地消失在走廊的盡頭。

只要閉上眼，世界就是無意識的樂園。
這就是我要的幸福。
只要這樣就好。
